# Virt-Manager Tips

## Enable 3D acceleration for Linux guest in Virt-Manager

1. Open Setting for VM
<img width="198" height="797" alt="image" src="https://github.com/user-attachments/assets/9c053488-58cd-446e-b37a-6c01f17b1aa3" />

2. Go to Video settings, choose Virtio and check 3D acceleration
<img width="605" height="547" alt="image" src="https://github.com/user-attachments/assets/e974782d-5709-4c9d-94cf-fce90a345579" />

3. Go to Display Splice and choose options as on picture
<img width="508" height="350" alt="image" src="https://github.com/user-attachments/assets/41fd4967-4118-489c-948f-0d869524e9dd" />

4. Reboot VM

## Enable Auto resizing of VM window for Linux guest in Virt-Manager

1. Click View
<img width="320" height="277" alt="image" src="https://github.com/user-attachments/assets/888abba0-0519-4db7-ab63-09cdf12e43b7" />

2. Choose Scale Display, check Always and mark Auto resize VM with window
